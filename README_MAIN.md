## 🚀 Quick Start

### Clone the Repository
```bash
git clone https://github.com/Zoro4TW/ML-models-Finlatics.git
cd ML-models-Finlatics
```

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run Notebooks
```bash
jupyter notebook
```

Navigate to the project folder of interest and open the `.ipynb` file.
